JChat
=====

Simple Java RMI chat.

## Main features

	1.	Two types of users (normal and Admin)
	2.	Minimal Client
	3.	Authentication Server
	4.	Public and private chat
	5.	SSL connection
	6.	Policy file for client
	7.	Remote error management
